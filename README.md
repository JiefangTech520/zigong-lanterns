## 🚀 Quick Start

### Option 1: Direct File Open
Simply open `index.html` in your browser:
```bash
D:/AI/自贡彩灯官网/index.html
```

### Option 2: Local Server
Using Node.js:
```bash
npx serve D:/AI/自贡彩灯官网
```

Using Python:
```bash
cd D:/AI/自贡彩灯官网
python -m http.server 8080
```

Then visit: http://localhost:8080

## 🎨 Design Highlights

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Deep Black | `#0a0a0f` | Primary background |
| Gold Primary | `#d4af37` | Brand accent, buttons |
| Gold Light | `#ffd700` | Highlights, hover states |
| Text Primary | `#ffffff` | Main text |
| Text Secondary | `rgba(255,255,255,0.7)` | Secondary text |

### Typography
- **Chinese:** Noto Serif SC, Noto Sans SC
- **English:** Playfair Display, Inter, Cinzel

### Key Features
- Glass morphism with `backdrop-filter: blur(20px)`
- Golden gradient accents
- Smooth scroll behavior
- Intersection Observer for scroll animations
- CSS Grid and Flexbox layouts
